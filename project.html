Cloud-based-file-sharing-system

    Tools and Technologies Used:

    • A Cloud-Based File Sharing System enables users to upload, store, and share files securely via the internet.
    • It ensures file access from any device and location using scalable cloud infrastructure.

    Tools and Technologies Used:

    • Frontend: HTML, CSS, JavaScript (with optional Bootstrap or Tailwind CSS)
    • Backend: Python with Flask Framework
    • Database: SQLite or AWS RDS (MySQL/PostgreSQL)
    • Cloud Storage: AWS S3 (Simple Storage Service)
    • Authentication: Flask-Login or JWT (optionally AWS Cognito)

    How the Project Works:

    • User Registration & Login: Users register or log in. Credentials are securely validated using hashing.
    • File Uploading: Logged-in users upload files through a web form, stored in AWS S3. File metadata is stored in the database.
    • File Sharing: The system generates a unique shareable link with access permissions.
    • Access & Download: The link allows file download or preview after verifying access permissions.
